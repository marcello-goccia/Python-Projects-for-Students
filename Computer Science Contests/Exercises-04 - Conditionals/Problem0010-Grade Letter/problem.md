## Title
Grade Letter

## Description
Given a student's score, determine their grade letter based on the following scale:

-90 and above: "A"
-80 to 89: "B"
-70 to 79: "C"
-60 to 69: "D"
-Below 60: "F"

## Input Description
An integer representing the student's score (0 to 100).

## Output Description
Print the grade letter corresponding to the score.

## Input Samples
85

## Output Samples
B
