## Title
Divide and Round Down

## Description
Given two integers, divide the first integer by the second using integer division and return the result.

## Input Description
Two lines containing two integers, a and b.

## Output Description
Print the result of a // b.

## Input Samples
10
3

## Output Samples
3


## Test Cases

Hidden Test Cases

Input:
15 4
Output:
3

Input:
99 10
Output:
9

Input:
5 2
Output:
2
