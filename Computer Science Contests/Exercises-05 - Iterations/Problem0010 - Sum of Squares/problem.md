## Title
Sum of Squares

## Description
Write a program that calculates the sum of the squares of all integers from 1 to n.
(e.g. for n = 3, calculate 1^2 + 2^2 + 3^3 = 14)

## Input Description
A single positive integer

## Output Description
The sum of squares of integers from 1 to n.

## Input Samples
3

## Output Samples
14
