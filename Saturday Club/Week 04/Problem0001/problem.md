## Title
Grade Classification

## Description
Given a student's score, classify the grade according to the following rules:
根据学生的分数，按照以下规则分类成绩：

Score >= 90: "A"
分数 >= 90: "A"

Score >= 80 and < 90: "B"
分数 >= 80 且 < 90: "B"

Score >= 70 and < 80: "C"
分数 >= 70 且 < 80: "C"

Score >= 60 and < 70: "D"
分数 >= 60 且 < 70: "D"

Score < 60: "F"
分数 < 60: "F"

## Input Description
A single integer between 0 and 100 inclusive, representing the student's score.
一个0到100之间的整数，表示学生的分数。

## Output Description
A single character representing the grade.
一个字符，表示成绩等级。

## Input Samples
85

## Output Samples
B
