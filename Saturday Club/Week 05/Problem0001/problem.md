
#### **Title:** Calculate Area of a Rectangle

**Description:**  
Write a program that reads the length and width of a rectangle and calculates its area.

**Input Description:**  
Two integers separated by a space, representing the length and width of the rectangle.

**Output Description:**  
The area of the rectangle.

**Input Samples:**  
`5 10`

**Output Samples:**  
`50`

**Test Cases:**

**Input 1:**  
`3 7`  
**Output 1:**  
`21`

**Input 2:**  
`8 8`  
**Output 2:**  
`64`

**Input 3:**  
`6 9`  
**Output 3:**  
`54`

#### **Title:** Matrix Addition

**Description:**  
Write a program that reads two 2x2 matrices and outputs their sum.

**Input Description:**  
Four lines. The first two lines represent the first matrix, and the next two lines represent the second matrix. Each line contains two integers.

**Output Description:**  
The resulting 2x2 matrix after adding the two matrices, each element on a new line.

**Input Samples:**  
`1 2`  
`3 4`  
`5 6`  
`7 8`

**Output Samples:**  
`6 8`  
`10 12`

**Test Cases:**

**Input 1:**  
`2 3`  
`4 5`  
`6 7`  
`8 9`  
**Output 1:**  
`8 10`  
`12 14`

**Input 2:**  
`0 0`  
`0 0`  
`1 1`  
`1 1`  
**Output 2:**  
`1 1`  
`1 1`

**Input 3:**  
`-1 -1`  
`-2 -2`  
`3 3`  
`4 4`  
**Output 3:**  
`2 2`  
`2 2`
